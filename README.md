# 🚀 Sitio Web Synapleads - Sistema Completo con Panel de Administración

## 📋 **RESUMEN EJECUTIVO**

**Synapleads** es un sitio web profesional especializado en la generación de leads para seguros de gastos finales con sistema de **live transfer inmediato**. El proyecto incluye un sitio web completo optimizado para conversión y un **panel de administración sin código** para gestión de contenido.

### **🎯 Objetivo Principal:**
Conectar leads calificados con agentes especializados en seguros de gastos finales en tiempo real, maximizando las conversiones mediante una experiencia de usuario optimizada y un sistema de gestión de contenido intuitivo.

## 📊 **ARQUITECTURA DEL SISTEMA**

### **🗂️ Estructura de Archivos**
```
synapleads-website/
├── admin/                          # Panel de administración
│   ├── index.html                  # Página principal del admin
│   ├── admin-styles.css            # Estilos del panel
│   ├── admin-auth.js              # Sistema de autenticación
│   ├── admin-main.js              # Funcionalidad principal
│   └── admin-editors.js           # Editores específicos
├── data/                          # Archivos JSON de contenido
│   ├── config.json                # Configuración global
│   ├── home-content.json          # Contenido página principal
│   ├── landing-content.json       # Contenido landing seguros
│   ├── gracias-content.json       # Contenido página gracias
│   ├── legal-content.json         # Contenido páginas legales
│   └── navigation.json            # Navegación del sitio
├── css/                           # Estilos del sitio web
│   ├── styles.css                 # Estilos principales
│   ├── landing.css                # Estilos específicos landing
│   ├── gracias.css                # Estilos página gracias
│   └── legal.css                  # Estilos páginas legales
├── js/                            # JavaScript del sitio
│   ├── main.js                    # JavaScript global
│   ├── content-loader.js          # Carga dinámica de contenido
│   ├── home.js                    # JavaScript página principal
│   ├── landing.js                 # JavaScript landing page
│   ├── gracias.js                 # JavaScript página gracias
│   └── legal.js                   # JavaScript páginas legales
├── images/                        # Recursos gráficos
│   └── logo-synapleads.png        # Logo corporativo
├── index.html                     # Página principal
├── landing-seguros.html           # Landing page seguros
├── gracias.html                   # Página de agradecimiento
├── privacidad.html                # Política de privacidad
├── terminos.html                  # Términos de uso
├── README.md                      # Este archivo
└── ADMIN-README.md                # Manual del panel admin
```

### **🔄 Flujo de Datos**
1. **Contenido** → Archivos JSON en `/data/`
2. **Carga** → JavaScript dinámico desde JSON
3. **Edición** → Panel admin actualiza JSON
4. **Visualización** → Sitio web refleja cambios inmediatamente

- **Desktop:** Layout completo con sidebar y columnas
- **Tablet:** Adaptación a pantallas medianas
- **Mobile:** Stack vertical, menú hamburguesa, botones touch-friendly
